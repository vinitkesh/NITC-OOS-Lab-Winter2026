# NITC OOS Lab Winter 2026

This repository contains the code and resources for the Object-Oriented Software (OOS) Lab at NITC for the Winter 2026 semester. The lab focuses on designing and implementing object-oriented software systems using Java.

Taken by Prof. Joe Cherri Ross in Winter 2026

## Directory Structure
- `PDFs` : Contains PDF files related to the lab, assignments, Evaluation questions.
- `EVAL` : Contains Java files for the evaluation questions and tests.
- `ASSGN` : Contains Java files for the Nth assignment corresponding to the PDF for nth assignment in `PDFs`.

Table for assignments and topics covered:

| Assignment Number | Topics Covered | 
|-------------------|----------------|
| 1                 | Basic Java - Arthematic,loops, arrays,etc |
| 2                 | Input in java,  String manipualtion, Java Arrays, recursion |
| 3                 | Basics of class, Exception handling, ArrayList, HashSet |
| 4                 | Sorting, LinkedHashSet, HashMap, PriorityQueue, Iterator, TreeMap |
| 5                 | Inheritance, access specifiers, super |
| 6                 | Java generics, Inheritance, class associations |
